Ayuda Clima App v.1.0

Esta App te permite consultar el clima de una cuidad en particular utilizando la API de OpenWeatherMap

Para poder consutar el clima, primero debeá Agregar una cuidad y luego seleccionar la Ciudad en cuestión desde la página Clima.
